# Access Control Policy

Access to production systems requires multi-factor authentication (MFA) for all
employees and contractors. Role-based access control (RBAC) is enforced, and
access grants are reviewed quarterly by the security team.

Privileged access (database admin, infrastructure admin) requires a documented
approval from a manager and expires automatically after 90 days unless renewed.

All access to customer data is logged, including read access, and logs are
retained for 12 months in a write-once storage system.
